# Clipboard Manager Application

## 概要
カスタマイズ可能なクリップボード/定型文管理アプリです。
ホットキーを使用して、定型文の呼び出し（`Alt+V`）や新規保存（`Ctrl+Shift+S`）が可能です。

## ファイル構成
- **main.py**: アプリの起動、ホットキーの監視、イベント制御を行うメインプログラム。
- **ui.py**: 画面のデザイン（一覧画面、保存画面）やボタンの動作などを記述。
- **data_handler.py**: データの保存・読み込み（`templates.json`への書き込み）を担当。
- **build.py**: 実行ファイル（.exe）を作成するためのスクリプト。
- **templates.json**: 【重要】保存した定型文データが入っています。

## 今後のアップデート手順
機能を追加したり修正したりしたい場合は、以下の手順で行います。

1.  **ソースコードの修正**:
    *   VSCodeなどで `.py` ファイルを編集して機能を変更します。
2.  **動作確認**:
    *   ターミナルで `python main.py` を実行し、変更が正しく動くか確認します。
3.  **アプリ化（exe更新）**:
    *   ターミナルで `python build.py` を実行します。
    *   `dist` フォルダ内の `ClipboardManager.exe` が新しいものに上書きされます。

## バックアップについて
このフォルダ（ワークスペース）全体を保存してあれば大丈夫です。
特に重要なのは以下の2つです。
1.  **ソースコード** (`.py` ファイル群): これがないと修正・再ビルドができません。
2.  **templates.json**: あなたが登録した定型文データです。

## 開発環境の復元
新しいPCなどで開発を再開する場合は、Pythonをインストールした後、以下を実行してライブラリを入れてください。
```bash
pip install customtkinter keyboard pyperclip pyinstaller
```
